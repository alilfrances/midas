# Security Policy

## Reporting a Vulnerability

Please report security issues privately through GitHub Security Advisories when available, or by opening a minimal public issue that asks for a private reporting channel without disclosing exploit details.

Include the affected version or commit, reproduction steps, expected impact, and any known mitigations. I will acknowledge valid reports and coordinate fixes as quickly as practical for a small personal project.

## Supported Versions

Only the latest `main` branch is supported.
