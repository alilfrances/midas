# Contributing

Thanks for helping improve Midas.

## Development

Midas uses Python 3.10+ and the standard library only.

Run the test suite from the repository root:

```sh
python -m unittest discover -s tests -v
```

## Pull Requests

- Keep changes focused.
- Add or update tests for behavior changes.
- Do not add runtime dependencies unless the tradeoff is discussed first.
- Keep hook output small and deterministic.
